# Sender-Rubika
A Rubika Sender bot that is activated on the account and used for groups.

# Installing on Windows
```
pip install rubpy
git clone https://github.com/silent-mimi/Sender-Rubika.git
cd Sender-Rubika
```
# Run on Windows
```
python sender-rubika.py
```


# Installing on termux
```
pkg install python
pkg install git
pip install rubpy
git clone https://github.com/silent-mimi/Sender-Rubika.git
cd Sender-Rubika
```
# Run on termux
```
python sender-rubika.py
```
